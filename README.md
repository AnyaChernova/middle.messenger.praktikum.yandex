### Полезные команды
- `npm install` — установка зависимостей,
- `npm run dev` — запуск версии для разработчика,
- `npm run start` — команда для запуска проекта на 3000 порту,
- `npm run build` — сборка проекта.

```sh
npm install
```

```sh
npm run dev
```

```sh
npm run start
```

```sh
npm run build
```
